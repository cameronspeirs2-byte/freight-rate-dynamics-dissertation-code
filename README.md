# freight-rate-dynamics-dissertation-code
R and Colab scripts for dissertation analysis of freight-rate dynamics

This repository contains the code used for my undergraduate economics dissertation on freight-rate dynamics on the Far East Asia–US West Coast route.

The repository includes R scripts for the econometric and machine-learning analysis, alongside a Google Colab/Python script used to clean and construct the AIS-derived operational variables.

The dissertation examines the determinants and predictability of freight-rate movements using monthly econometric models and daily machine-learning models. The analysis combines freight-rate data with operational, weather, macroeconomic, cost and trade-policy variables.
